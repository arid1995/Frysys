# Frysys
REVOLUTIONARY GAME EVAAA
Ваня - карта, прорисовка
Фарид - Бегать прыгать, взаимодействие с объектами
Серый - движение карты, генерация
Задания до 12 мая:
Первостепенные:
1. Переписать собственные объекты в стиле ООП, обеспечить полиморфизм
2. Создать врагов и ништяки (вроде выхода)
3. Написать взаимодействие игрока с врагами
4. Написать меню
5. Отладить работу движка (исключить провалы и застревания)
6. Добавить условия завершения игры (смерть, выигрыш)
7. Добавить звуковые эффекты (для начала просто музычка)
Второстепенные:
1. Создать хотя бы еще один уровень
2. Прописать движение бэкграунда (параллакс)
3. Сделать событийные звуковые эффекты (смерть, удар, прыжок и т.д.)
4. Встроить многопоточность хоть куда-нибудь
